
# QKLockView 
 最新版本的swift 写的手势密码控件，使用简单方便。 xib、纯代码皆可。 

<br>
## Requirements
* Xcode 7.3 or higher
* iOS 8.0 or higher 
* 最新的swift，swift对8.1以下的支持的还不好


## 本demo的介绍 <br> 1. QKLockView.swift 是最重要的类，可以拿到你的项目里用   2. 有个CALayer的分类，新手可以学习下，swift里extension的写法
